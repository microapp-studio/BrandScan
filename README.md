# BrandScan

BrandScan — MVP приложения для секонда, винтажа и ресейла.

Что умеет:
- открывает камеру телефона;
- делает снимок ярлыка;
- принимает фото из галереи;
- принимает цену в секонде;
- отправляет фото и цену в ИИ;
- возвращает бренд, тип вещи, новую цену, б/у цену, риск оригинальности и решение: брать / осторожно / не брать / нужно больше фото;
- устанавливается на телефон как PWA.

## Важно

На чистом GitHub Pages ИИ не заработает, потому что API-ключ нельзя хранить в открытом фронтенде.

Для полной работы используй Vercel.

## Как запустить на Vercel

1. Создай новый репозиторий на GitHub.
2. Загрузи все файлы из архива в корень репозитория.
3. Зайди на vercel.com.
4. Import Project → выбери этот репозиторий.
5. В настройках проекта добавь Environment Variable:

OPENAI_API_KEY = твой ключ OpenAI API

6. Deploy.

После этого приложение будет работать по ссылке Vercel.

## Файлы

- index.html — интерфейс приложения
- api/analyze.js — backend-функция для ИИ
- manifest.json — PWA manifest
- sw.js — service worker
- icon-192.png — иконка
- icon-512.png — иконка
- package.json — настройки проекта
- vercel.json — настройки Vercel

## Предупреждение

BrandScan не является профессиональной экспертизой подлинности. Приложение помогает быстро оценить вещь, но окончательное решение остается за пользователем.
